Управление Материалами

Этот раздел предоставляет функциональность для управления материалами в вашей программе.

  • Добавить запись: Нажмите на кнопку "Добавить запись", чтобы добавить новый материал.
  • Удалить запись: Выделите материал в таблице и нажмите кнопку "Удалить запись" для удаления выбранного материала.
  • Редактировать запись: Выделите материал в таблице и нажмите кнопку "Редактировать запись" для внесения изменений.
  • Сохранить изменения: После внесения изменений в запись нажмите кнопку "Сохранить изменения".
  • Найти: Введите текст в поле поиска и нажмите кнопку "Найти" для поиска материалов по заданным критериям.
  • Экспорт в Excel: Нажмите на эту кнопку, чтобы экспортировать данные о материалах в файл Excel.
